# 🚀 Link Generator - Quick Start Guide

## 30-Second Setup

### For Teachers/Admin

1. **Open Admin Panel**
   ```
   https://yourdomain.com/attendance-link-admin.html
   ```

2. **Create a Link**
   - Select: Semester, Subject, Teacher Name
   - Click: "Generate Link" button

3. **Share the Link**
   - Copy the full URL from green box
   - Paste in WhatsApp/Email to students
   - Link works for exactly 5 minutes

---

## For Students

1. **Receive Link from Teacher**
   ```
   https://yourdomain.com/attendance-link-generator.html?token=xyz123abc
   ```

2. **Click the Link**

3. **Fill Your Details**
   - Name (required) ✍️
   - Roll Number (required) 🎓
   - Email (optional) 📧
   - Mobile (optional) 📱

4. **Watch the Timer**
   ```
   ⏱️ 05:00 (5 minutes)
   ⏱️ 04:30 
   ⏱️ 01:00 (Warnings start here!)
   ⏱️ 00:00 (EXPIRED)
   ```

5. **Click Submit**

6. **See Success** ✅

## Important Points

### ⚠️ Timer Rules
- ❌ Cannot extend time
- ❌ Cannot pause timer
- ✅ Must submit BEFORE 00:00
- ✅ Link autom-ically disables at 5 minutes

### 🔒 Security Rules
- One submission per device
- Cannot submit twice from same link + device
- Device ID stored in browser cache
- Different browser = different device

### 👨‍🏫 Teacher Instructions
1. Generate link just before class
2. Share on class WhatsApp/Email
3. Delete/disable link when attendance closes
4. Download report anytime for records

### 👨‍🎓 Student Instructions
1. Click link IMMEDIATELY when teacher shares
2. Fill your name and roll number
3. Check the timer (don't wait till last second)
4. Click submit before timer reaches 00:00

## Troubleshooting

### "Link Expired"
- Link is older than 5 minutes
- Ask teacher for a new link

### "Already Submitted"
- Your device already submitted
- Use different device or wait 5+ minutes

### Nothing Works
- Refresh the page
- Check internet connection
- Try different browser
- Clear browser cache
